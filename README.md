# ⚡ Fluxio

> SaaS de gestion des créances, des paiements et de la trésorerie.
> Pour commerçants sénégalais. Mobile-first.

**Promesse :** Suivez vos paiements, récupérez vos créances, maîtrisez votre trésorerie.

---

## Stack

- **Backend** : Node.js / Express
- **Database** : PostgreSQL (Railway)
- **Frontend** : Next.js 14
- **Paiement** : PayDunya (Wave + Orange Money)
- **Repo** : GitHub (terangaprestigegroup-star/recouvpro)
